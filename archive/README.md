# Архив вспомогательных материалов

Здесь лежат файлы, **не нужные** для открытия сайта в браузере (`index.html`), но полезные как ссылки, база URL и черновики ТЗ.

| Путь | Содержимое |
|------|------------|
| **`stubs/*.md`** | Короткие указатели на разделы `../PROJECT-DIRECTIVES.md` (совместимость имён файлов). |
| **`references.json`** | Опциональная база ссылок для модулей (сайт её не подгружает). |
| **`tz-texts/*.txt`** | Сырые импорты ТЗ. |
| **`uav-student-supplementary-backup.zip`** | Копия `stubs/` + `references.json` + `tz-texts/` — можно хранить вне проекта или на диске. |

## Восстановить «как раньше» в корень

1. Скопировать `archive/stubs/*.md` → в корень проекта (поверх не сработает — файлы уже перемещены; при необходимости извлечь из ZIP).
2. При желании вернуть `references.json` и папку `tz-texts/` в корень из `archive/`.

Полные тексты правил и история изменений по-прежнему в **`PROJECT-DIRECTIVES.md`** в корне.
